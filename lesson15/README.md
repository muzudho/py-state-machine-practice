# Lesson15

`transition_conf.py` だけあれば、状態遷移がちゃんとつながっているか検証できるはずです。  
graphviz を使って可視化しましょう  

# Set up

📖 　[Graphviz Download](https://graphviz.org/download/)

```shell
pip install graphviz
```

# Run

```shell
python.exe -m lesson15.main
```

Output:  

`lesson15-graphs.png`
